HTML DOM

Introducción

Cuando se carga un sitio web, el navegador crea un modelo de objeto del documento (DOM) del mismo

Dicho DOM está constituido de varios objetos como se describe en el gráfico

Con este modelo, JavaScript puede crear HTML dinámico, con todo lo que puede hacer:

  • Javascript puede cambiar todos los elementos HTML
  • Javascript puede cambiar todos los atributos HTML
  • Javascript puede cambiar todos los estilos CSS
  • Javascript puede remover elementos y atributos del HTML
  • Javascript puede añadir nuevos elementos y atributos HTML
  • Javascript puede reaccionar a todos los eventos HTML
  • Javascript puede crear nuevos eventos HTML
